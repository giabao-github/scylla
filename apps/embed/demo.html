Embed Sandbox

Scylla Widget Demo

A cleaner local sandbox for testing script embedding, switching positions, and verifying the widget lifecycle before shipping it into a real site.

What this page does

The demo auto-loads the widget once using a hard-coded organization ID, then lets you reinitialize, show, hide, and destroy it while watching the live runtime state.

Runtime status tracking

Important note

The widget loader script below auto-initializes on page load. That means the first visible state is already mounted, and the initialize action behaves like a reinitialize control.
